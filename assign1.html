Base Building Assignment

1. what is the Inernet? Explain in your own words.

Ans = The Internet is a vast network that connects computers all over the world. Through the Internet, people can share information and communicate from anywhere with an Internet connection.

2. what is a web browser? Explain in your own words.

Ans = A web browser is a software application used to access and display content on the World Wide Web. The web browser is the primary tool for accessing the Internet, and it allows users to view web pages, images, videos, and other types of content.

3. What is browser engine? Explain in your own words.

Ans = A browser engine, also known as a rendering engine or layout engine, is the core software component of a web browser that interprets and renders web content such as HTML, CSS, and JavaScript.

4. What is static site? Explain in your own words.

Ans = A static site is a type of website that is built using only HTML, CSS, and JavaScript files. Overall, static sites offer a simple and efficient way to build websites, and they are a good option for those who prioritize speed, security, and ease of maintenance over dynamic functionality.

5. What is dynmaic site? Explain in your own words.

Ans = A dynamic site is a type of website that generates content on the fly using server-side scripts and databases. Dynamic sites use programming languages such as PHP, Ruby, Python, or JavaScript to generate HTML, CSS, and JavaScript files on the server-side. These files are then sent to the user's browser, where they are displayed as web pages.
